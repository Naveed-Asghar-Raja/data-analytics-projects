# Data Analytics Projects

Proyectos de análisis de datos enfocados en construcción, marketing y web analytics.
Data analytics projects focused on construction, marketing and web analytics.

- **Database:** PostgreSQL (Supabase)
- **Visualization:** Power BI
- **Programming:** Python
- **Spreadsheets:** Excel
- **Construction Software:** ISTRAM-ISPOL (BIM for Infrastructure),Autocad
- **Spanish Cost Database:** Presto, BEDEC/ITeC

Analysis of website traffic data including sessions, users, conversions and bounce rate.

### 2. Marketing Analytics

### 3. Construction Analytics *(Coming Soon)*
